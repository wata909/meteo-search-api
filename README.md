#  @naro/agroenv-search-api

## development

```shell
$ git clone git@github.com:geolonia/naro-agroenv-search-api.git
$ cd naro-agroenv-search-api
$ cp .envrc.sample .envrc
$ vim .envrc # fill the values
$ yarn # or `npm install`
$ npm test
```

### start localserver

```shell
$ npm start
```

### build js library

```shell
$ npm run build:lib
```

## deploy

```shell
$ npm run deploy:dev
```
